Подключение

// Вариант 1 — динамический импорт (рекомендуется)
window.AIWidgetConfig = {
  apiKey: "gsk_ВАШ_GROQ_КЛЮЧ",
  title: "Поддержка",
  systemPrompt: "Ты ассистент интернет-магазина.",
  position: "bottom-right",  // или "bottom-left"
};
import("https://your-domain.com/widget.js");

// Вариант 2 — тег <script>
// <script>window.AIWidgetConfig = { apiKey: "..." };</script>
// <script src="https://your-domain.com/widget.js"></script>

Как начать

  1. Зайдите на console.groq.com и создайте бесплатный API ключ.
  2. Скопируйте widget.js на ваш сервер (или используйте CDN).
  3. Добавьте две строки на любую страницу (конфиг + скрипт).
  4. Готово! Кнопка чата появится в нижнем углу экрана.

Публичный API

// Доступен после загрузки виджета
window.AIWidget.open();          // Открыть
window.AIWidget.close();         // Закрыть
window.AIWidget.logout();        // Выйти
window.AIWidget.getUser();       // Текущий юзер
window.AIWidget.setConfig({     // Обновить настройки
  apiKey: "новый_ключ"
});
